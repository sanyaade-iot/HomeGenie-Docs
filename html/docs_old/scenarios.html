Scenarios and Lighting Effects

Interactive Recording

To create scenarios and lighting effects, we can take advantage of the Record Macro functionality. From the Control section, access the Actions menu to start recording as shown in the following image.

From now on, every action taken (eg. controlling lights/appliances from the user interface or with a remote control), will be recorded.
By default, it will be put 1 second delay between each command, though we can also switch to other kinds of timings.
One of these is called Mimic. When Mimic is selected, HomeGenie will replicate exactly the delay we put between commands while recording.

By clicking the "Save" option in the footer bar, we'll save the current interactive recording session.

Wizard Scripts

After saving an interactive recording session, a new "Wizard Script" (macro), containing all performed commands, will be created and opened in the Program Editor.
Here we can add/remove commands, change the name, the description and the associated automation group.

A Wizard Script requires no programming knowledge and it can either be created from the "Record Macro" option, as just discussed, or from the automation section itself by selecting the "Add new program" option from its Actions menu.

Manual Script Execution

A Wizard Script can be added like a module to a group (see "Using HomeGenie" for a quick start about configuring groups and modules).

After adding it to a group, it will shows up as a button in the Control menu. Tapping this button it will execute the script.
A coloured led, on the left side of the button, will tell us when the script is running (green light) or when it's idle (yellow light).

A script can also be executed from the Program Editor's Actions menu by selecting the "Run" option (see here).

Automated Script Execution

Beside the manual execution, we can automatically execute a script whenever some conditions are met.
By tapping the "Add Condition" button, we will be able to set all the "Program Conditions" that will trigger the execution of the script.

In the above picture we have an example where we want to turn on two lights whenever the motion is detected from a sensor and then switch them off after 1 minute.
