Using HomeGenie

Once installed we need to configure the system. This can be achived in few steps.

Configuring: easy as 1, 2, 3

1. Enabling Control Interfaces

From the main menu go to Configure->System and Interfaces.

Here we can activate our control Interfaces (eg. X10 / Z-Wave controllers).

2. Enabling APPs and Features

Going back to the Configure menu we can access the Automation section. Here we'll find, grouped into categories, some Automation Program Plugins (also called APPs). Program plugins add functionalities and features to HomeGenie. We can enable/disable each APP as download and import new ones. By clicking on a APP in the list, we can eventually configure its options.

3. Adding Groups and Modules

Dimmers, switches, sensors and all other automation devices (modules) can now be organized into groups. From "Configure->Groups and Modules" section we can add new groups (eg. Living Room, Kitchen, etc...) and associate our modules to them.

Controlling

Now that everything has been setup, we can go back to main menu and access the Control section, where we can monitor and control our modules.
