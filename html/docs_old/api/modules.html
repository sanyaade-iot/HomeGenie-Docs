.Each

Iterate through each module in the current selection.

.Each (  fn(module) )Returns: ModulesManager

module
Type: ModuleHelper
A reference to each module will be passed to the fn delegate. If the delegate returns true, the iteration will break.
Example snippet:
    var total_watts_load = 0D;
    Modules.WithParameter("Meter.Watts").Each( (module) => {
      total_watts_load += module.Parameter("Meter.Watts").DecimalValue;
      return false; // continue iterating
    });
    Program.Notify("Current power load", total_watts_load + " watts");

.IsOn

Boolean read only property.

.IsOnReturns: boolean

This property will return true if at least one module in the current selection is on, otherwise it will return false.

.IsOff

Boolean read only property.

.IsOffReturns: boolean

This property will return true if all modules in the current selection are off, otherwise it will return false.

.Level

Property of type double used to get/set "Status.Level" parameter of each selected module (averge value is returned when more than one module is selected).

.LevelReturns: double

Level property is meant to be a percentage value (0-100).
Example snippet:
    Modules.WithFeature("EnergyManagement.EnergySavingMode").Level = 40;

.Command

Select an API command to be executed for selected modules. To perform the selected command, Execute or Set method must be invoked.

.Command( api_command )Returns: ModulesManager

api_command
Type: string
A string containing any valid HomeGenie api command. It can be a standard command or a custom one (see RegisterDynamicAPI from ProgramHelper class). Standard commands are:
  • Control.On
  • Control.Off
  • Control.Level
  • Control.Toggle
  • Control.ColorHSB (valid for Philips Hue)
Example snippet:
	// turn on all modues of type "Light"
    Modules.OfDeviceType("Light").Command("Control.On").Execute();
    // set all dimmers to 50%
    Modules.OfDeviceType("Dimmer").Command("Control.Level").Set("50");

.Execute

.IterationDelay

Used before a command .Set or .Execute, it will put a pause after performing the selected command for each module.

.IterationDelay (  seconds )Returns: ModulesManager

seconds
Type: double
Pause in seconds to be put between each command.
Example snippet:
    // 100ms delay between each command
    Modules
      .OfDeviceType("Dimmer")
      .Command("Control.Level")
      .IterationDelay(0.1)
      .Set(40);
